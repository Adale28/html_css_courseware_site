Lesson Objectives: CSS Introduction

This lesson plan outlines the objectives, prerequisites, and exclusions of the lesson teaching students how to apply basic Cascading Style Sheets (CSS) to format and style elements on a web page.

Students will be able to:

  • define Cascading Style Sheets (CSS),
  • explain the purpose of CSS in web development,
  • distinguish between inline, internal, and external CSS,
  • identify and use basic CSS properties (e.g., color, font-size, background-color),
  • apply CSS to style HTML elements,

Prerequisites

Students should already be familiar with:

  • the structure and function of basic HTML elements,
  • how to write and test simple HTML code using a live editor.

Exclusions

Students will not cover:

  • advanced CSS features such as animations, transitions, or media queries,
  • the use of external stylesheets in web development environments beyond online live editors.
